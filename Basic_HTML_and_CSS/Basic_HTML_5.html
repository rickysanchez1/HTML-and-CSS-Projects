Character Entities and how to use them.

Character Entities are reserved characters in HTML and must be replaced with alternative characters. There are 2 ways you can type entity names. With numbers or with letters.
I needed to type "&amp" in order to display the & next to the entity name or numbers. Such as "&#60" for <

Below are some of the entities you need to use in order to use the characters in your code.

Character Description Name Number
< Less than &lt; &#60
< Greater than &gt; &#62
& Ampersand &amp; &#38;
" Double quote &quot; &#34;

Text Shadow property

The text shadow property is used by adding "text-shadow" to the CSS selector to style your element.
